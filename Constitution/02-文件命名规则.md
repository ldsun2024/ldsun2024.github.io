# 文件命名规则

## 文件名规则
- 只允许：`a-z`、`A-Z`、`0-9`、`-`（横线）、`.`（点号）
- 不允许：中文、空格、逗号、冒号、问号、引号等特殊符号
- 不允许中文标点（`，` `。` `？` `：` `"` `"` 等）

## 命名格式
```
{编号}-{简短关键词}.md
```

## 文章标题（front matter）
- `title` 字段可以用完整中文，不受文件名限制
- 标题与文件名无关，页面显示的是 `title` 的内容

## 生成后必做验证
1. `hugo` 构建不报错
2. 文章列表页能正常显示新文章
3. 点击新文章能正常打开内容（非404）
